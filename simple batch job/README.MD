1. Simple Batch Job (Processing Dummy Data)

You can simulate processing by working with a list of data instead of files. For example, let's say you want to process some numbers and apply an operation like squaring them.

What It Does:

The process_item() function squares each number.

The batch_job() function processes a list of numbers (in this case, 1 to 15).
